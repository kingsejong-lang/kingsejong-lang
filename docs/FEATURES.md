## 1. 핵심 언어 기능

### 1.1 데이터 타입

#### F001: 정수 (Integer)
- **우선순위**: CRITICAL
- **상태**: 🚧 파싱만 완료, 평가 미구현
- **설명**: 정수 리터럴 및 연산
- **예제**:
  ```
  나이 = 15
  개수 = 100
  음수 = -42
  ```
- **구현 범위**:
  - ✅ 정수 리터럴 파싱
  - ❌ 정수 평가 및 저장
  - ❌ 오버플로우 처리
  - ❌ 큰 정수 지원 (BigInt)

#### F002: 실수 (Float)
- **우선순위**: HIGH
- **상태**: ❌ 미구현
- **설명**: 부동소수점 리터럴 및 연산
- **예제**:
  ```
  파이 = 3.14159
  온도 = -273.15
  ```
- **구현 범위**:
  - ❌ 실수 리터럴 파싱
  - ❌ 실수 평가 및 저장
  - ❌ 정수-실수 자동 변환
  - ❌ 부동소수점 정밀도 처리

#### F003: 문자열 (String)
- **우선순위**: CRITICAL
- **상태**: ❌ 미구현
- **설명**: 문자열 리터럴, 연산, 메서드
- **예제**:
  ```
  이름 = "김철수"
  인사 = "안녕하세요!"
  여러줄 = "첫째 줄
  둘째 줄"
  ```
- **구현 범위**:
  - ❌ 문자열 리터럴 파싱 (따옴표, 큰따옴표)
  - ❌ UTF-8 한글 문자열 처리
  - ❌ 문자열 연결 (`"안녕" + "하세요"`)
  - ❌ 이스케이프 시퀀스 (`\n`, `\t`, `\"`)
  - ❌ 문자열 인덱싱 (`이름[0]`)
  - ❌ 문자열 슬라이싱 (`이름[0:2]`)
  - ❌ 문자열 메서드 (길이, 분리, 찾기 등)

#### F004: 불린 (Boolean)
- **우선순위**: HIGH
- **상태**: 🚧 파싱만 완료
- **설명**: 참/거짓 값
- **예제**:
  ```
  참 = true
  거짓 = false
  ```
- **구현 범위**:
  - ✅ 불린 리터럴 파싱 (`true`, `false`)
  - ❌ 불린 평가
  - ❌ 논리 연산 (`and`, `or`, `not`)
  - 📝 한글 키워드 (`참`, `거짓`) 고려

#### F005: Null/없음 타입
- **우선순위**: HIGH
- **상태**: ❌ 미구현
- **설명**: 값이 없음을 나타내는 타입
- **예제**:
  ```
  값 = 없음
  ```
- **구현 범위**:
  - ❌ null/없음 리터럴
  - ❌ null 검사
  - 📝 키워드: `없음`, `null`

#### F006: 배열/리스트
- **우선순위**: HIGH
- **상태**: ❌ 미구현
- **설명**: 여러 값을 저장하는 순서 있는 컬렉션
- **예제**:
  ```
  숫자들 = [1, 2, 3, 4, 5]
  이름들 = ["철수", "영희", "민수"]
  혼합 = [1, "안녕", true, 3.14]
  ```
- **구현 범위**:
  - ❌ 배열 리터럴 구문 (`[...]`)
  - ❌ 인덱싱 (`숫자들[0]`)
  - ❌ 슬라이싱 (`숫자들[1:3]`)
  - ❌ 배열 메서드 (추가, 제거, 정렬 등)
  - ❌ 다차원 배열

#### F007: 딕셔너리/객체 (선택)
- **우선순위**: MEDIUM
- **상태**: 📝 계획됨
- **설명**: 키-값 쌍 저장
- **예제**:
  ```
  학생 = {
    "이름": "김철수",
    "나이": 15,
    "학년": 3
  }
  ```

### 1.2 연산자

#### F010: 산술 연산자
- **우선순위**: CRITICAL
- **상태**: 🚧 파싱만 완료
- **설명**: 수학 연산
- **연산자**:
  - `+` (덧셈)
  - `-` (뺄셈)
  - `*` (곱셈)
  - `/` (나눗셈)
  - `%` (나머지) - 미구현
  - `**` (거듭제곱) - 미구현
- **예제**:
  ```
  합 = 5 + 3          # 8
  차 = 10 - 4         # 6
  곱 = 6 * 7          # 42
  몫 = 20 / 4         # 5
  나머지 = 17 % 5     # 2
  제곱 = 2 ** 8       # 256
  ```

#### F011: 비교 연산자
- **우선순위**: CRITICAL
- **상태**: 🚧 파싱만 완료
- **설명**: 값 비교
- **연산자**:
  - `==` (같음)
  - `!=` (다름)
  - `<` (작음)
  - `>` (큼)
  - `<=` (작거나 같음) - 미구현
  - `>=` (크거나 같음) - 미구현
- **예제**:
  ```
  나이 = 15
  만약 (나이 >= 18) {
    출력("성인")
  }
  ```

#### F012: 논리 연산자
- **우선순위**: HIGH
- **상태**: 🚧 부분 구현 (NOT만)
- **설명**: 불린 값 연산
- **연산자**:
  - `!` (NOT) - 파싱 완료
  - `&&` 또는 `and` (AND) - 미구현
  - `||` 또는 `or` (OR) - 미구현
- **예제**:
  ```
  성인 = 나이 >= 18
  학생 = true

  만약 (성인 and 학생) {
    출력("성인 학생입니다")
  }
  ```

#### F013: 할당 연산자
- **우선순위**: CRITICAL
- **상태**: 🚧 기본만 구현
- **설명**: 변수에 값 할당
- **연산자**:
  - `=` (할당) - 파싱만 완료
  - `+=`, `-=`, `*=`, `/=` (복합 할당) - 미구현
- **예제**:
  ```
  x = 5
  x += 3    # x = x + 3
  ```

### 1.3 제어 구조

#### F020: 조건문 (if/else)
- **우선순위**: CRITICAL
- **상태**: 🚧 파싱만 완료
- **설명**: 조건에 따른 분기
- **문법**:
  ```
  만약 (조건) {
    // 참일 때 실행
  } 아니면 {
    // 거짓일 때 실행
  }
  ```
- **예제**:
  ```
  나이 = 15
  만약 (나이 >= 18) {
    출력("성인입니다")
  } 아니면 {
    출력("미성년자입니다")
  }
  ```
- **구현 범위**:
  - ✅ if 문법 파싱
  - ✅ else 문법 파싱
  - ❌ 조건 평가 및 실행
  - ❌ else if (여러 조건)

#### F021: While 반복문
- **우선순위**: CRITICAL
- **상태**: ❌ 미구현
- **설명**: 조건이 참인 동안 반복
- **문법**:
  ```
  반복 (조건) {
    // 반복 실행
  }
  ```
- **예제**:
  ```
  개수 = 0
  반복 (개수 < 5) {
    출력(개수)
    개수 = 개수 + 1
  }
  # 출력: 0 1 2 3 4
  ```

#### F022: For 반복문
- **우선순위**: HIGH
- **상태**: ❌ 미구현
- **설명**: 범위 또는 컬렉션 반복
- **문법 1**: 범위 반복
  ```
  반복 (i = 0; i < 10; i = i + 1) {
    출력(i)
  }
  ```
- **문법 2**: 컬렉션 반복 (고급)
  ```
  이름들 = ["철수", "영희", "민수"]
  반복 (각각 이름들에서 이름) {
    출력(이름)
  }
  ```

#### F023: Break/Continue
- **우선순위**: MEDIUM
- **상태**: 📝 계획됨
- **설명**: 반복문 제어
- **예제**:
  ```
  반복 (i = 0; i < 10; i = i + 1) {
    만약 (i == 5) {
      중단  # break
    }
    만약 (i % 2 == 0) {
      계속  # continue
    }
    출력(i)
  }
  ```

### 1.4 함수

#### F030: 함수 정의
- **우선순위**: CRITICAL
- **상태**: ❌ 미구현
- **설명**: 재사용 가능한 코드 블록 정의
- **문법**:
  ```
  함수 함수이름(매개변수1, 매개변수2) {
    // 함수 본문
    return 반환값
  }
  ```
- **예제**:
  ```
  함수 덧셈(a, b) {
    return a + b
  }

  함수 인사하기(이름) {
    return "안녕하세요, " + 이름 + "님!"
  }
  ```

#### F031: 함수 호출
- **우선순위**: CRITICAL
- **상태**: ❌ 미구현
- **설명**: 정의된 함수 실행
- **예제**:
  ```
  결과 = 덧셈(5, 3)
  출력(결과)  # 8

  메시지 = 인사하기("철수")
  출력(메시지)  # 안녕하세요, 철수님!
  ```

#### F032: Return 문
- **우선순위**: CRITICAL
- **상태**: 🚧 파싱만 완료
- **설명**: 함수에서 값 반환
- **예제**:
  ```
  함수 제곱(x) {
    return x * x
  }
  ```

#### F033: 기본 매개변수 (선택)
- **우선순위**: LOW
- **상태**: 📝 계획됨
- **예제**:
  ```
  함수 인사하기(이름 = "손님") {
    출력("안녕하세요, " + 이름 + "님!")
  }

  인사하기()        # 안녕하세요, 손님님!
  인사하기("철수")  # 안녕하세요, 철수님!
  ```

#### F034: 가변 인자 (선택)
- **우선순위**: LOW
- **상태**: 📝 계획됨
- **예제**:
  ```
  함수 합계(...숫자들) {
    총합 = 0
    반복 (각각 숫자들에서 숫자) {
      총합 = 총합 + 숫자
    }
    return 총합
  }

  출력(합계(1, 2, 3))        # 6
  출력(합계(1, 2, 3, 4, 5))  # 15
  ```

#### F035: 클로저 (고급)
- **우선순위**: MEDIUM
- **상태**: 📝 계획됨
- **설명**: 외부 변수를 기억하는 함수
- **예제**:
  ```
  함수 카운터() {
    개수 = 0
    함수 증가() {
      개수 = 개수 + 1
      return 개수
    }
    return 증가
  }

  카운트 = 카운터()
  출력(카운트())  # 1
  출력(카운트())  # 2
  ```

#### F053: Try/Catch (선택)
- **우선순위**: LOW
- **상태**: 📝 계획됨
- **설명**: 예외 처리 구문
- **예제**:
  ```
  시도 {
    결과 = 10 / 0
  } 잡기 (에러) {
    출력("에러 발생:", 에러)
  }
  ```

### 2.2 수학

#### F110: 기본 수학 함수
- **우선순위**: MEDIUM
- **상태**: ❌ 미구현
- **함수 목록**:
  - `절대값(x)` - abs
  - `제곱근(x)` - sqrt
  - `거듭제곱(x, y)` - pow
  - `올림(x)` - ceil
  - `내림(x)` - floor
  - `반올림(x)` - round
  - `최댓값(...)` - max
  - `최솟값(...)` - min
- **예제**:
  ```
  출력(절대값(-5))      # 5
  출력(제곱근(16))      # 4
  출력(거듭제곱(2, 10)) # 1024
  출력(최댓값(1, 5, 3)) # 5
  ```

#### F111: 삼각 함수 (선택)
- **우선순위**: LOW
- **함수**: `sin`, `cos`, `tan`, `asin`, `acos`, `atan`

#### F112: 상수

### 2.3 문자열

#### F120: 문자열 메서드
- **우선순위**: HIGH
- **상태**: ❌ 미구현
- **메서드**:
  - `길이(문자열)` - 문자열 길이
  - `대문자(문자열)` - 대문자 변환
  - `소문자(문자열)` - 소문자 변환
  - `분리(문자열, 구분자)` - 문자열 분리
  - `합치기(배열, 구분자)` - 배열 합치기
  - `찾기(문자열, 패턴)` - 패턴 찾기
  - `바꾸기(문자열, 이전, 이후)` - 문자열 교체
  - `잘라내기(문자열, 시작, 끝)` - 부분 문자열
- **예제**:
  ```

  ```

#### F130: 배열 메서드
- **우선순위**: HIGH
- **상태**: ❌ 미구현
- **메서드**:
  - `길이(배열)` - 배열 길이
  - `추가(배열, 값)` - 끝에 추가
  - `삽입(배열, 인덱스, 값)` - 특정 위치 삽입
  - `제거(배열, 인덱스)` - 특정 위치 제거

- **예제**:
