## 개요

KingSejong은 한국어의 문법적 특성을 활용한 혁신적인 프로그래밍 언어입니다. Phase 1, 2, 3를 완료하여 프로덕션 수준의 언어로 발전했습니다.

## 주요 기능

### 1. 조사 기반 메서드 체이닝

한국어의 조사를 활용한 자연스러운 메서드 호출:

```ksj
배열을 정렬한다
데이터를 변환한다
결과를 출력한다
```

### 2. 함수형 프로그래밍

고차 함수와 함수형 메서드:

```ksj
숫자들을 걸러낸다(함수(x) { 반환 x % 2 == 0 })
       을 변환한다(함수(x) { 반환 x * x })
       을 축약한다(0, 함수(누적, 현재) { 반환 누적 + 현재 })
```

### 3. 모듈 시스템

```ksj
가져오기 "stdlib/math"
가져오기 "stdlib/array"
```

### 4. 자동 메모리 관리

Mark & Sweep 가비지 컬렉터로 순환 참조 자동 해제.

### 5. 바이트코드 컴파일

AST → Bytecode 컴파일로 성능 향상.

### 6. 완전한 문서

4,200+ 줄의 종합 문서 세트.
