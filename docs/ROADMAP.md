**핵심 아이디어**: 한국어 시제를 활용한 직관적인 비동기 프로그래밍

#### 개념 설계
```ksj
# 현재 시제: 동기 실행
파일을 읽는다("data.txt")

# 미래 시제: 비동기 실행
파일을 읽을것이다("data.txt")  # Promise 반환
결과 = 파일을 읽을것이다("data.txt")를 기다린다  # await

# 과거 시제: 완료된 작업
작업이 완료되었다  # 완료 조건 확인
```
