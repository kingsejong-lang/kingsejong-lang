**영향**:
- 튜토리얼의 많은 예제가 타입 없는 문법을 사용
- 파일 실행 시 파서 에러 발생

**조치**:
- 섹션 3에 타입 명시 권장 안내 추가
- 주요 예제에 타입 키워드 추가
- 전체 예제는 추가 작업 필요

**권장 사항**:
1. 파서를 수정하여 타입 추론 지원
2. 또는 LANGUAGE_REFERENCE.md 수정하여 타입 필수로 명시
3. 튜토리얼 전체를 타입 명시 문법으로 업데이트

### ⚠️ 이슈 2: `타입()` 함수 오용

**문제**:
- 튜토리얼에서 `타입(숫자)`를 숫자→문자열 변환으로 사용
- 실제로는 `타입(25)` → `"정수"` (타입 이름 반환)

**예시**:
```ksj
# 튜토리얼 코드 (잘못됨)
나이 = 25
출력("나이: " + 타입(나이))  # 출력: "나이: 정수" ❌

# 올바른 코드
나이 = 25
출력("나이:", 나이)  # 출력: "나이: 25" ✅
```

**조치**:
- 일부 예제 수정 시작
- 전체 수정 필요

**권장 사항**:
1. 숫자→문자열 변환 함수 추가 (`문자열()` builtin)
2. 튜토리얼 전체에서 `타입()` 오용 제거
3. `출력()` 다중 인자 사용 패턴으로 변경

### ⚠️ 이슈 3: 주석 파싱 문제

**테스트 결과**:

**권장 사항**:
